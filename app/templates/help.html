What is Google Dorking?

Google Dorking (also known as Google hacking) is a technique that uses advanced Google search operators to find specific information that might not be easily accessible through regular browsing. This tool automates the process of running multiple dorking queries against a target domain.

Note: This tool is intended for authorized security testing and reconnaissance only. Ensure you have proper authorization before scanning any domain.

How to Use
  1. Enter Domain: Enter the target domain (e.g., example.com)
  2. Select Categories: Choose specific categories or leave blank for all
  3. Start Scan: Click "Start Scan" to begin the reconnaissance
  4. Monitor Progress: Watch the scan progress in real-time
  5. View Results: Once complete, review and export the results
Scanning Categories
Category Description
Subdomain Enumeration Find subdomains of the target domain
Exposed FTP Discover publicly accessible FTP services
Exposed Documents Find exposed PDF, Word, Excel and other documents
Exposed Git Locate exposed Git repositories and configuration
Directory Listings Find exposed directory listings and sensitive files
Code Leaks Search for code leaks on external code sharing sites
Cloud Storage Discover exposed cloud storage buckets
API Documentation Find API endpoints and documentation
Web Servers Identify web server types (Apache, Nginx, etc.)
CMS Detection Find WordPress, Drupal, Joomla installations
Common Google Search Operators
Operator Example Purpose
site: site:example.com Limit results to a specific domain
intitle: intitle:"index of" Find pages with specific title
inurl: inurl:admin Find pages with specific URL pattern
intext: intext:"error" Find pages with specific text
filetype: filetype:pdf Find specific file types
ext: ext:php Find files with specific extension
cache: cache:example.com View Google's cached version
link: link:example.com Find pages linking to a domain
Important Notes
  • Authorization: Always obtain proper authorization before scanning any domain
  • Responsible Use: Use this tool responsibly and ethically
  • Rate Limiting: Google may rate limit your searches - queries have delays built in
  • False Positives: Results may include false positives - verify all findings
  • Legal Implications: Unauthorized access to systems is illegal
  • Privacy: Respect user privacy and data protection laws
